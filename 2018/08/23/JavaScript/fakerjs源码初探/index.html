FakerJs源码解读

发表于 2018-08-23 | 分类于 JavaScript ， NodeJs

我们写项目的时候经常要写单元测试，通常来说，我们的单元测试都是黑盒测试。比如我们写了一个 “统计注册用户数量” 的功能，为这个功能写单元测试的时候一般不需要考虑用户注册功能，这很容易理解。测试这个统计功能的一个难题就是模拟一些注册用户的数据？nodejs 有丰富的第三方模块方便我们做黑盒测试，这里主要介绍如何使用faker.js来生成测试时需要的数据

安装

简单使用

从例子就可以知道，faker.js 的使用其实很简单，首先设置一个语言环境(默认英文)，然后就可以使用它生成各种各样的 “假数据” 了。

本地化

Faker 从 v2.0.0 版本开始就支持多个地方，默认语言环境设置为英语，设置一个新的地区很简单:

个人本地化包

v3.0.0 开始，faker.js 支持增量加载 locale，默认 faker 包括所有地区的数据，在生产环境中，可能想只包含特定一组区域设置的设置数据，写法也很简单，只加载本地语言环境

下面具体介绍常用函数功能以及源码解析

helpers

我的理解 helpers 是把一组不明确归类，但是业务使用场景上经常会使用到的一些函数放在一起

randomize(array)

传递数组，返回某个随机元素，底层调用的是 Faker.random.arrayElement(array)

slugify(string)

删除所有非字母数字字符，从字符串中删除所有特殊字符具体看例子

  • 源码

函数接受一个字符串，现将所有空格替换为 “-“，接下来经过正则找出特殊的字符替换为空字符。

  • e.g

replaceSymbolWithNumber(string, symbol)

用随机数替换 symbol 符号，当未指定 symbol 参数，默认替换 “#” 符号

###

replaceSymbols(string)

用符号解析字符串，并适当地替换它们，当字符串出现 “#” 符号将会被替换为 0到10 的随机数， 出现 “?” 符号将会被替换为 A到Z 的字符

###

shuffle(array)

传递一个数组，返回打乱顺序之后的数组

address

一个地址一般包括 国家、省、市、区、县、街道、邮政编码、经纬度 这些信息，faker 也提供了专门生产这类书籍的函数

国家

country()

国家，随机从 faker.definitions.address.country 选一个值返回

countryCode()

国家代码，随机从 faker.definitions.address.country_code 选一个值返回

省份

state()

省份，随机从 faker.definitions.address.state 选一个值返回

stateAbbr()

省份简写，随机从 faker.definitions.address.state 选一个值返回

城市

city()

城市名称

  • 源码

函数接受一个 0 ~ 3 之间的整数参数，对应 formats 元素索引位置，代表返回结果的格式

  • e.g

cityPrefix()

城市前缀，随机从 faker.definitions.address.city_prefix 选一个值返回

citySuffix() - 城市后缀

随机从 faker.definitions.address.city_suffix 选一个值返回

县、郡

county()

随机返回 faker.definitions.address.county 的一个值

街道

streetName()

返回街道名称, 内容包括: faker.name.lastName() + faker.address.streetSuffix()

streetAddress(useFullAddress)

返回一个随机的本地化街道地址

  • 源码

解析一下以上源码：faker.random.number(2) 返回一个 0~2 的整数，address 包括：3~5 随机个数字、streetName 街道名称，如果调用函数的时候传递了useFullAddress 参数并且为 true ，返回的结果后面还加上一个 secondaryAddress

  • e.g

streetPrefix()

街道前缀，该函数随机从 faker.definitions.address.street_prefix 返回一个值

streetSuffix()

街道前缀，该函数随机从 faker.definitions.address.street_suffix 返回一个值

zipCode(format)

邮编

  • 源码

函数最终调用一个 replaceSymbols 方法处理 format

  • e.g

longitude()、latitude()

随机返回一个经纬度值，保留4位小数

phone

phoneNumber(format: string)

随机生成一个电话号码或者手机号码，可以指定格式

phoneNumberFormat(phoneFormatsArrayIndex: number)

格式化的电话号码

phoneFormats()电话格式

###-########、###########

internet

avatar()

返回一张网络图片，头像

email(firstName, lastName, provider)

返回邮箱地址

userName(firstName, lastName)

名字

protocol()

随机返回某个Internet协议

url()

随机返回一个faker.internet.protocol() + ‘://‘ + faker.internet.domainName()组合的结果

domainName()

随机返回一个faker.internet.domainWord() + “.” + faker.internet.domainSuffix()组合的结果

domainSuffix()

随机返回 faker.definitions.internet.domain_suffix 的某个元素

domainWord()

结果为：faker.name.firstName().replace(/([\~#&*{}/:<>?|\”‘])/ig, ‘’).toLowerCase()

userAgent()

color(baseRed255, baseGreen255, baseBlue255)

根据传递的rgb值返回一个颜色值

random

random.number(options)

  • 源码

直接看参数[ options ]，如果我们希望得到一个 0~n 的随机整数，我们只需要调用函数的时候传递一个整数 n；如果希望得到一个 m~n 的整数，我们需要传递一个Object 类型的参数，该参数包括 “min”、”max”、”precision” 三个值。max 和 min 参数比较好理解，precision表示对应的精度，很明显当精度为 1，表示去 min~max直接的随机整数

  • e.g

arrayElement(array)

接受一个数组，随机返回数组中的某个元素

objectElement(object, field)

从对象中选择一个随机元素，选择值而不是键。这个函数用于创建各种可能性的自定义列表

  • 源码

源码很简单，当 field 参数不传或者不等于 “key”，则返回 object 参数某个随机 key 对应的值；而如果 field 等于 “key” 则直接返回随机的某个 key 名

  • e.g

uuid()

boolean()

image()

相当于 Faker.image.image()

locale()

随机返回一个语言环境

alphaNumeric()

α数字
