MomentJs使用笔记

发表于 2018-08-25 | 分类于 JavaScript ， NodeJs

以前看过写Java的时候，经常有定义一个DateUtils的工具类，它提供了很多日期处理的静态方法。在nodejs里面，你就可以不用担心日期时间的处理了，使用momentjs类库可以很方便转化日期。momentjs很简单，然而以前写代码用到还是会经常搜下哪个函数是我想要的，能得到我所期待的效果。难得今天我也是有空，按照自己习惯的使用当成笔记记下来，下次我就可以不用专门搜索了

定义

momentjs 时间处理类库，用于解析、校验以及显示日期
