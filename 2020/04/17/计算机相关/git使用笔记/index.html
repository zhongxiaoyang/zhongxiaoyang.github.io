提交文件到暂存区（cache）

git add命令会把未跟踪的文件提交到暂存区，未跟踪的文件是无法提交的，因为没有纳入仓库的版本控制管理中。一般这个命令在我们正常的理解里面完成三种功能：

  • 将未跟踪的新文件添加到版本控制
  • 将未跟踪的新内容添加到版本控制
  • 标记合并冲突状态为已解决
